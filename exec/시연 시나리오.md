#### 1. 소셜로그인 
- 소셜 로그인을 합니다.

#### 2. 텍스트 입력 
- 사용자 텍스트 인풋을 입력합니다
  - 상황 : 여자친구와 200일
  - 대상 : 
  - 메시지 : 
- 만들기 버튼 클릭

#### 3. 이미지 생성
- 꽃다발이 만들어질 때까지 대기 (발표자가 기술 설명) 약 40초에서 1분 소요

#### 4. 추천 및 직접 꽃 선택하여 재생성
- 맨 첫 꽃과 두번째 꽃을 변경
- 맨 첫 번째 꽃은 추천으로
- 두 번째 꽃은 직접 고르기
  - 색깔별로 되어있는 것 확인시키고 선택
- 재생성은 시간관계상 생략한다고 말하고 확인버튼 누름
  - 만들기 버튼 클릭

#### 5. 이미지 다운로드
- 다운로드 받음

#### 6. 다른 사용자 이미지 검색 및 다운로드
- 검색에 들어가서 무한 스크롤만 보여줌 
- 다른 꽃 아무거나 다운로드
- 인기순이 반영된다고 설명

#### 7. 지도 API
- 클릭해서 주변 꽃집 확인
- 꽃집 정보 클릭해서 보여주기
- 만약 안되면 핸드폰에서는 정확히 된다고 언급
